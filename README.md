# Satellite Wi-Fi Research Website

This is a group project website built using HTML, CSS. Our chosen topic is Satellite Wi-Fi which critically appraise past, present, and future application of this technology with particular emphasis on their impact on society in general, and on individual, while incorporating current and potential impacts.

It showcases our research which includes a short introduction in abtract section, its risk, opportunities, choices.


## Features
- Custom typography
- Hero section with large headline
- Project categories
- Clean modern design

- CSS
